# Auto Scoring Darts System (ESP32 + Mechanical Switches)

## About This Project

This project is an automated dart scoring system built using mechanical switches and an ESP32 microcontroller.

The objective is to detect dart impacts on the board using a grid of mechanical switches positioned behind the scoring zones. When a dart hits a specific area, the corresponding switch is triggered. The ESP32 reads the activated switch, determines the score based on its position, and updates the total automatically in real time.

This system aims to:

- Automate dart score calculation  
- Reduce manual scoring errors  
- Provide real-time score updates  
- Offer a low-cost embedded alternative to commercial electronic dart boards  

The project combines embedded systems programming, hardware design, switch matrix implementation, debouncing techniques, and real-time scoring logic.
